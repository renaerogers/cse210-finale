# cse210-student-overview 
This is the repository for the CSE 210 final project 

This repository contains the code for this project. They are arranged as follows:

* `csharp` - This project is written using C#.
* `final` - This project contains all of the files containing the code for the cse210 Eternal Quest Goal Tracker.
* `design` - This project is a demonstration of the student's understanding of C# with a focus on Polymorphism.
